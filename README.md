# Tasks Records

A lightweight and secure task management system built with Vanilla PHP and MySQL. This project demonstrates a complete CRUD (Create, Read, Update, Delete) workflow designed for efficiency and ease of use.

## Key Features
* **Full CRUD Functionality**: Easily create, view, edit, and remove tasks from the system.
* **Dynamic Background**: Features a personalized UI with a custom background image and semi-transparent glassmorphism effects.
* **Responsive Design**: Built using Bootstrap 5 to ensure a clean look across all devices.
* **Automated Timestamps**: Real-time tracking of when tasks are created and updated using MySQL timestamps.
* **Modern UI Icons**: Integrated FontAwesome for intuitive action buttons.

## Technical Specializations
* **Backend**: PHP (Vanilla)
* **Database**: MySQL
* **Frontend**: Bootstrap 5, Custom CSS
* **Icons**: FontAwesome 6.x
